| |     This commit adds Flask as a framework for the website and implements routes for the home and chat pages. It also adds functionality to load and save the chat history from a JSON file, and a route for generating responses from the chatbot API. The generate_text function gets the user's input from the request form, loads the chat history from the file, adds the input to the chat history, gets a response from the chatbot API using the get_response function, saves the updated chat history to the file, and returns the response as a JSON object. This allows users to view their chat history and continue their conversation with the chatbot. The Flask integration also allows for more advanced functionality to be added in the future, such as user authentication and session management.
